# Fits Logit or Probit model.

Fits Logit or Probit model.

## Usage

``` r
acro_glm(formula, data, family)
```

## Arguments

- formula:

  The formula specifying the model.

- data:

  The data for the model.

- family:

  Decide whether to fit a logit or probit model.

## Value

Regression Results Wrapper
