# Nursery Database

This dataset is originated from a hierarchical decision model created to
evaluate applications for nursery schools.

## Usage

``` r
nursery_data
```

## Format

A data frame with columns: A data frame with 12960 rows and 9 columns:

- parents:

  Parents' occupation

- has_nurs:

  Child's nursery

- form:

  Form of the family

- children:

  Number of children

- housing:

  Housing conditions

- finance:

  Financial standing of the family

- social:

  Social conditions

- health:

  Health conditions

- recommend:

  The ranking of applications for nursery schools

## Source

https://www.openml.org/search?type=data&status=active&id=26&sort=runs

## Examples

``` r
data(nursery_data)
```
